This is a Quick Review on Methods and Classes

Class

A class describes the content of the objects that belong to it.

  • Think of it as an object blueprint.
  • When we create a new object, we are instantiating the class (literally creating a new instance of the class)

Object

An object is an instance of a class. The structure of the object is defined by its class


Method

A method is a procedure associated with a message and an object. In other words - a method contains an action that an object can perform.

  • Methods provide the interface that other classes use to access and modify the data properties of an object.
